# Markdown to WeChat

一个面向微信公众号后台粘贴场景的 Markdown 渲染工具，开箱即用，无需后端服务。

## ✨ 功能特性

- 📝 **完整 Markdown 支持** - 基于 markdown-it，支持 CommonMark 语法、嵌套列表、复杂表格、脚注、任务列表、HTML 混排
- 🎨 **6 套精美主题** - 默认蓝、海洋蓝、森林绿、日落橙、薰衣草紫、深色主题，可视化卡片选择
- 📄 **多草稿管理** - 支持新建、重命名、复制、删除草稿，实时自动保存
- 🛠️ **图片增强工具** - 圆角调节、间距控制、4 种图注样式、边框/阴影开关
- 🔒 **微信兼容层** - 自动过滤公众号不支持的样式，确保复制内容完美显示
- 📊 **发布前检查** - 字数统计、阅读时间、本地图片检测、不兼容样式提示
- 💾 **配置导入导出** - 所有设置支持备份恢复，跨设备同步
- 📋 **一键复制导出** - 直接复制到公众号后台，支持导出 HTML 文件

## 🚀 快速开始

### 方式一：直接使用
1. 下载 `index.html` 文件到本地
2. 用浏览器直接打开即可使用，无需任何其他依赖

### 方式二：本地服务
```bash
# 启动本地 HTTP 服务
python3 -m http.server 8080
# 访问 http://localhost:8080
```

### 方式三：GitHub Pages
1. Fork 本仓库
2. 进入仓库 Settings → Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" 分支，根目录
5. 保存后等待部署完成，即可通过 `https://你的用户名.github.io/Markdown-to-WeChat` 访问

## 📖 使用说明

1. 在左侧编辑区输入 Markdown 内容
2. 右侧实时预览公众号排版效果
3. 在设置面板选择主题、调整样式
4. 点击「复制到剪贴板」
5. 直接粘贴到公众号后台编辑器即可

## 🎯 适用场景

- 微信公众号排版
- 知乎专栏排版
- 小红书图文排版
- 其他富文本编辑器内容导入

## 🛡️ 安全说明

- 所有数据都保存在浏览器本地 localStorage，不会上传到任何服务器
- 无后端服务，纯前端实现，隐私安全有保障
